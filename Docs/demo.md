# Demo Workflow

Step 1
User opens the HerShield mobile app.

Step 2
User presses the SOS button.

Step 3
The mobile app sends:

- victim image
- GPS location
- user information

to the drone server.

Step 4
Drone receives the request and launches.

Step 5
AI identifies the victim using face recognition.

Step 6
Drone streams live video.

Step 7
Emergency contacts and authorities are notified.
